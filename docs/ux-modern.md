# Modern UX Pack

## Features
- Product images: drag & drop uploads, reorder by drag, and mobile-friendly move up/down controls.
- Price tags: print queue reorder (drag + move buttons) in Products.
- Command palette: Cmd/Ctrl+K global search + quick actions.
- Bulk actions: consistent selection toolbar in Products, Suppliers, Inventory, and Purchase Orders.

## Shortcuts
- Cmd/Ctrl+K: open command palette.
- Esc: close the palette or dialogs.
- Arrow Up/Down: navigate palette results.
- Enter: open selected result or run action.

## Notes
- Drag & drop has button fallbacks for mobile and accessibility.
- Bulk actions always show selection count and clear action.
